Análise Exploratória de Dados de Filmes

Análise exploratória de dados (EDA) de filmes, com dados de 1980 a 2020, para identificar fatores que influenciam o faturamento, como orçamento, gênero e produtora. Projeto MVP desenvolvido para a sprint de Análise de Dados e Boas Práticas da pós-graduação em Ciência de Dados e Analytics da PUC-Rio.

Objetivos

- Identificar fatores que impactam o faturamento dos filmes
- Analisar padrões e distribuições nos dados
- Explorar relações entre variáveis relevantes

Questões de Pesquisa
- Filmes com maior orçamento geram maior faturamento?
- Existem gêneros mais lucrativos que outros?
- Filmes produzidos por determinadas empresas apresentam melhor desempenho financeiro?

Principais Análises
- Relação entre orçamento e faturamento
- Faturamento médio por gênero
- Empresas com maior desempenho financeiro
- Distribuição e análise de outliers

Principais Insights
- O faturamento dos filmes apresenta alta variabilidade, com presença de outliers
- Filmes com maior orçamento tendem a gerar maior faturamento, embora não seja uma regra absoluta
- Determinados gêneros e produtoras se destacam em termos de desempenho financeiro
- O sucesso na indústria cinematográfica é concentrado em poucos filmes
